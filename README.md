⚡ Motor Control Center (MCC) Professional Designer
Industrial Electrical Panel Design Tool

A Python + Streamlit-based engineering application that automates the design process of Motor Control Center (MCC) panels used in industrial electrical systems.

This software helps engineers perform motor calculations, component selection, cable sizing, panel costing, and documentation generation in a fast and structured workflow.

The goal of this project is to simplify electrical panel design and reduce manual engineering calculations by providing an easy-to-use graphical interface.

🚀 Features
⚙ Motor Engineering Calculations

Motor full load current calculation

Supports kW / HP-based input

Designed for industrial 3-phase motors

🔌 Automatic Feeder Design

Supports common MCC feeder types:

DOL (Direct On Line) Starter

Star-Delta Starter

VFD Feeder

🧠 Intelligent Component Selection

Automatic component recommendation based on motor rating.

Components included:

MCCB / Circuit Breaker

Contactor

Overload Relay

Power Cable

Selection is based on engineering rules and standard sizing tables.

📏 Cable Sizing

Cable selection based on:

Motor current

Safety margin

Engineering cable rating tables

💰 Panel Cost Estimation

The tool estimates the MCC feeder's cost using predefined component prices.

Includes cost of:

MCC Panel

MCCB

Contactor

OLR

Cable

Control components

📊 Bill of Materials (BOM)

Automatically generates a Bill of Materials for the designed feeder.

Features:

Export BOM to Excel

Organised component list

Quantity and cost breakdown
